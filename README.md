# Audio-Fingerprinting

## Description
The  primary objective of this project was to develop a Basic song Recognition System like SHAZAM . It tajes few Audio Clips as queries , both as a batch of clips and individual , and Identifies  the song Using Spectrogram analysis and fingerprint matching . 
## Features 
### Generates Spectrograms
It computes the Spectrogram Matrix of a song and displays the  Spectrogram plots .
### Creates Constellation Maps 
It identifies the Local maximas  in the spectrogram , it plots the detected peaks both on the original spectrogram and as an independent scatter plot known as a constellation map. 
### Generates hashes for each song 
For every peak , it creates hashes , using  using the frequencies of the two peaks and the time difference.
### Build Fingerprint database of all songs 

Identifies song fromg given query clip 
Displays the intermediatary steps followed in song recognition
